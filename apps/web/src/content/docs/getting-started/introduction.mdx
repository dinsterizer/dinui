## Roadmap

- [x] `export default`
- [x] Can be installed via package managers -> easy to receive updates
- [x] Can be copied and used immediately without configuration files like `tsconfig.json`, `jsconfig.json`, `components.json` -> easy to modify
- [x] CLI installation
- [x] Multiple theme color with our `context color mode`
- [ ] Components for admin and dashboard
- [ ] Extends component variants without copy code into your project
- [ ] Figma `free`
