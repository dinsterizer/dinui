# Resizable

Accessible resizable panel groups and layouts with keyboard support.

import { Example } from '@web/components/example'

<Example path="ui/resizable-demo" />

## About

The `Resizable` component is built on top of [react-resizable-panels](https://github.com/bvaughn/react-resizable-panels) by [bvaughn](https://github.com/bvaughn).

## Installation

import { ComponentInstallationGuide } from '@web/components/component-installation-guide'

<ComponentInstallationGuide path="ui/resizable" />

## Usage

```tsx
import { ResizableHandle, ResizablePanel, ResizablePanelGroup } from '@dinui/react/resizable'
```

```tsx
<ResizablePanelGroup direction="horizontal">
  <ResizablePanel>One</ResizablePanel>
  <ResizableHandle />
  <ResizablePanel>Two</ResizablePanel>
</ResizablePanelGroup>
```

## Examples

### Vertical

Use the `direction` prop to set the direction of the resizable panels.

<Example path="ui/resizable-vertical" />

```tsx showLineNumbers {9}
import { ResizableHandle, ResizablePanel, ResizablePanelGroup } from '@dinui/react/resizable'

export default function Example() {
  return (
    <ResizablePanelGroup direction="vertical">
      <ResizablePanel>One</ResizablePanel>
      <ResizableHandle />
      <ResizablePanel>Two</ResizablePanel>
    </ResizablePanelGroup>
  )
}
```

### Handle

You can set or hide the handle by using the `withHandle` prop on the `ResizableHandle` component.

<Example path="ui/resizable-handle" />

```tsx showLineNumbers {11}
import { ResizableHandle, ResizablePanel, ResizablePanelGroup } from '@dinui/react/resizable'

export default function Example() {
  return (
    <ResizablePanelGroup direction="horizontal">
      <ResizablePanel>One</ResizablePanel>
      <ResizableHandle withHandle />
      <ResizablePanel>Two</ResizablePanel>
    </ResizablePanelGroup>
  )
}
```
